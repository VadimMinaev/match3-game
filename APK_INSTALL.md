# 📱 Установка игры "Шары" на Android

## ✅ Готовый APK файл
**Путь:** `Shary.apk` (в корне проекта)
**Размер:** 4.1 MB
**Версия:** Debug

## 🔧 Требования
- **Android версия:** 7.0+ (API 24+)
- **Свободное место:** 10-50 MB

## 📥 Способы установки

### Способ 1: Через компьютер (рекомендуется)
1. Подключите Android телефон к ПК через USB
2. Включите режим отладки (Settings → Developer Options → USB Debugging)
3. Откройте командную строку в папке с apk файлом
4. Выполните команду:
   ```bash
   adb install Shary.apk
   ```

### Способ 2: Через файловый менеджер
1. Скопируйте файл `Shary.apk` на телефон
2. Откройте файловый менеджер на телефоне
3. Найдите файл и нажмите на него
4. Подтвердите установку

### Способ 3: Через Android Studio
1. Откройте Android Studio
2. Выберите меню: Tools → Android → Android Device Monitor
3. Перетащите `Shary.apk` на эмулятор/устройство

## 🎮 Использование
Со своего ПК запустите в случае, если используется adb:
```bash
adb shell am start -n ru.vadmin.balls/.MainActivity
```

## 🐛 Решение проблем

### "Install failed: INSTALL_FAILED_INVALID_APK"
- Убедитесь, что APK файл не повреждён (4142825 байт)
- Попробуйте удалить предыдущую версию: `adb uninstall ru.vadmin.balls`
- Переустановите APK

### "USB Debugging not enabled"
1. Откройте Settings → About Phone
2. Нажмите 7 раз на "Build Number"
3. Вернитесь в Settings → Developer Options → USB Debugging

### Приложение не запускается
- Убедитесь, что Android версия 7.0+
- Попробуйте переустановить приложение
- Проверьте логи: `adb logcat`

## 🔄 Сборка новой версии
Для пересборки после изменений:
```bash
npm install
npx cap sync android
cd android
.\gradlew.bat assembleDebug
```

Готовый APK будет в: `android\app\build\outputs\apk\debug\app-debug.apk`

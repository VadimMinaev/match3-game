## 🎯 Специфика реализации

### Flexbox магия
```css
.game-container {
  display: flex;
  flex-direction: column;
  height: 100%;
}

.game-board {
  flex: 1;          /* Занимает оставшееся место */
  min-height: 0;    /* Важно для flex + aspect-ratio */
  aspect-ratio: 1/1;
}
```

### Модальное окно
```css
.modal {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  display: flex !important;  /* Переопределяет display: none */
}
```

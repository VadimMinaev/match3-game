## 📋 Как установить на Android

### Вариант 1: Через ADB (быстро)
```bash
# 1. Подключите телефон USB-кабелем
# 2. Включите USB Debugging на телефоне:
#    Settings → Developer Options → USB Debugging

# 3. Запустите в командной строке:
adb install Shary.apk

# 4. Игра будет установлена!
```

### Вариант 2: Через файловый менеджер (просто)
1. Скопируйте `Shary.apk` на телефон (через кабель или облако)
2. Откройте файловый менеджер на телефоне
3. Найдите `Shary.apk` и нажмите на него
4. Подтвердите установку в диалоге
5. Готово! 🎉

### Вариант 3: Через Android Studio (сложнее)
1. Откройте Android Studio
2. Tools → Android → Android Device Monitor
3. Перетащите файл на окно
4. Установкой завершится автоматически

## 🐛 Если что-то не работает

### Белый/черный экран при загрузке
```bash
# Посмотрите логи:
adb logcat | grep "Shary\|balls"
```

### "Cannot install", "INSTALL_FAILED_INVALID_APK"
```bash
# Удалите старую версию сначала:
adb uninstall ru.vadmin.balls

# Потом установите:
adb install Shary.apk
```

### Приложение зависает
- Переустановите APK
- Очистите кеш: Settings → Apps → Шары → Storage → Clear Cache
- Перезагрузите телефон

### Нет доступа к USB Debugging
- Settings → About Phone
- Нажмите 7 раз на "Build Number"
- Вернитесь, появится "Developer Options"
- Откройте его и включите USB Debugging

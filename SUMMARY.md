**Ключевые изменения:**

- **Уменьшены размеры:** шрифты, паддинги, гапы
- **Новые компоненты:** стили для модальки (`.modal`, `.modal-header`, `.modal-body`)
- **Скрыты блоки:** `.game-rules` и `.achievements-panel` → `display: none`

**Новые стили:**
- `.modal` с анимацией `fadeIn` и `slideUp`
- `.rules-btn` - кнопка открытия правил (36×36px круглая)
- `.header-top` - flex контейнер для заголовка и кнопки

**Всего изменено строк:** ~200 из 747

### 3. **game.js** 🎮
**Ключевые изменения:**

  - Закрытие по клику на `#close-rules`
  - Закрытие при клике вне модалки (на `#rules-modal`)
  - Закрытие по Escape (если модалка открыта)

**Строк изменений:** ~40 (из 1075)

### Шаги:
1. **npm install** - установка зависимостей ✓
2. **npx cap sync android** - синхронизация веб-ассетов ✓
3. **./gradlew assembleDebug** - компиляция приложения ✓
4. **Результат:** `android/app/build/outputs/apk/debug/app-debug.apk` ✓
5. **Копирование:** `c:\Code\match3-game\Shary.apk` ✓

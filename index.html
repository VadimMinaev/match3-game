📖 Правила игры

🎯 Основы

  • Соединяйте 3+ шарика одного цвета
  • Неудачный ход = штраф очков
  • Комбо = больше очков!

💣 Бонусы

  • 🟠 4 шарика = Линейная бомба (удаляет линию)
  • 🔴 5 шариков = Бомба 3×3 (взрывает куб)
  • 🌈 6+ шариков = Радужный шар (удаляет цвет)

⚡ Советы

  • Ищите совпадения в разных направлениях
  • Активируйте бонусы для больших взрывов
  • Планируйте вперёд!
